# Variations By Pedal

Keep pedal-oriented variations here, but use a flat folder layout.

## Structure
- One variation folder per run, for example:
  - `intentional-rat-YYYYMMDD-HHMMSS/`
  - `random-clean-boost-YYYYMMDD-HHMMSS/`
  - `manual-90s-20260323-182317/`
- Folder names can include pedal/type hints for humans, but there is no pedal-type directory taxonomy.
- Canonical metadata stays in each `snapshot.json` (`pedal_type`, `slug`, etc.).
